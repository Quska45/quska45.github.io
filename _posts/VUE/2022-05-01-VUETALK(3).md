## 환경 구성 스크립트
```
  // vue-cli 설치
  npm install -g @vue/cli

  // vue 프로젝트 생성
  vue create vue-electron-test

  // electron builder 추가
  vue add electron-builder

  // 실행
  npm run electron:serve
```
